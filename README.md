# Game Calendar

## Development

This application is integrated with Google Firebase. In order to build & run
it, You will need a file in the root of your repository named
`packages/app/firebase-config.json`. It should look like the following:

```json
{
  "__PROJECT_ID__": "firebase-project-id",
  "__API_KEY__": "firebase-api-key"
}
```

You can find the appropariate values in your project's settings.
